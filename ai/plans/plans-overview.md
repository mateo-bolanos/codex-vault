# AI Implementation Plans

Implementation plans, context digests, and task lists live under `ai/plans/` and are keyed by `task_slug`.

## Note types

- `[[ai/plans/{{TASK_SLUG}}-plan]]` – file-level implementation plan (`type: ai-impl-plan`).
- `[[ai/plans/{{TASK_SLUG}}-context]]` – short context digest (`type: ai-context-digest`).
- `[[ai/plans/{{TASK_SLUG}}-tasks]]` – implementation checklist (`type: ai-task-list`).
