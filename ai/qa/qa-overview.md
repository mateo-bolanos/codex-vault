# AI QA Sessions

QA / UX session notes live under `ai/qa/` and are typically generated by the User QA subagent.

## Conventions

- One note per persona-session: `ai/qa/<task_slug>-<persona>.md` or similar.
- Frontmatter uses `type: ai-qa-session`, `task_slug`, and `persona`.
- QA notes link back to the backlog task, plan, and workflow.
